Instructions

  1. See post WebVR coming to Servo
  2. Download experimental WebVR Servo build.
  3. Unpack the RAR archive into an appropriate location.
  4. Open Windows command line by clicking on Windows Start menu, and typing `cmd.exe` to launch the command line.
  5. Change to the directory where you unpacked the archive. eg: `cd c:\servo-webvr\`
  6. Run `servo.exe --resources-path resources webvr\room-scale.html`
  7. Click the `Enter VR` button.
